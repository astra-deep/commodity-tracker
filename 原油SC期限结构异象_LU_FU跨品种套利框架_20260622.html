二、SC为何"独自Contango"？—— 七层结构性解释

核心矛盾重述：全球原油市场处于供应危机→WTI/Brent深度Backwardation（近高远低，$35-40价差）。但SC（中国原油期货）近月→远月反而呈微幅Contango（近低远高，+9元/1.78%）。这种方向性背离是罕见的。

1 中国需求独立坍塌 —— 最根本原因

指标当前值正常/峰值偏离
地炼开工率55-60%70-75%5年最低
炼油利润-600至-1300元/吨+200至+500全线亏损
原油进口（5月）645万桶/日~1100万桶/日10年新低
战略库存消耗~100万桶/日正常0持续去库

逻辑链：炼厂不赚钱 → 不开工 → 不买现货原油 → 近月需求极弱 → 近月价格承压 → 形成Contango。 WTI/Brent定价的是"全球供应短缺"，SC定价的是"中国需求过剩库存"——两个市场讲的是完全不同的故事。

2 SC≠Brent：SC是中国局部市场，不是全球基准

SC-Brent贴水 = -11.6%（SC 507.2元 ≈ $70.6/桶 vs Brent $79.9/桶）

关键点：SC的定价权在中国境内市场。中国是全球最大原油进口国（日均~1100万桶），但SC只反映中国保税区供需，不反映全球。当中国需求疲弱时，SC比Brent更弱是合理的。如果SC也Backwardation，SC-Brent折扣会更大（可能到-15%~-20%），这反而不合理——因为仓储套利会限制折扣幅度。

实际上，SC的Contango是一种"国内供需贴水 + 仓储成本"的综合体现。近月弱是因为国内没人要，远月略高是因为仓储有成本+未来需求预期恢复。

3 进口配额+保税交割：物理套利被封堵

如果SC=Brent自由套利成立，那么SC Contango会被大量买SC卖Brent的套利盘抹平。但现实中：

  • 进口配额壁垒：中国原油进口实行配额制，地炼没有配额就买不了SC交割的实物原油，更不可能把SC低价油出口到国际市场
  • 保税交割限制：SC交割在指定保税库，外资参与有限。交割品级、地点、时间的限制远多于Brent
  • 仓储容量：SC保税库存高位，但新增仓单需要INE审批——供给不弹性
  • 结论：SC和Brent之间的套利通道是半封闭的，SC的Contango可以被"困"在国内

4 人民币汇率预期：隐性的"Contango放大器"

SC以人民币计价。如果市场预期人民币贬值（当前离岸6.78，经济疲弱+降息预期），那么：

  • 远月SC需要更高的人民币价格才能维持同等的美元价值
  • 假设Brent远期$72/桶，预期CNY从6.78贬至7.0，则SC远月 = $72 × 7.0 = 504元 → 若汇率不变则只有$72 × 6.78 = 488元
  • 汇率贬2.2%，SC远月自动"升水"16元/桶
  • 这16元中约一半是"真实的"供需Contango，另一半是"货币性的"汇率预期

5 供给恢复的"时间不对称"

全球原油供应的恢复是渐进式的，不同产油国恢复速度差异巨大：

产油国恢复时间增量对SC影响
沙特/阿联酋1-2周~200万桶/日近月供给改善→压近月
科威特1-1.5月~50万2-3月后供给改善
伊拉克3-6月~100万中远月供给恢复
伊朗3-6月+不确定性~150万远月最大变量

逻辑：近月面临沙特快速增产压力（压近月），但远月定价的是"全部恢复后"的正常市场（支撑远月）。 WTI/Brent的Backwardation之所以成立，是因为全球库存太低+即期需求强，即使供给恢复也需要6-12个月才能补库。 SC的Contango之所以成立，是因为中国库存仍高+即期需求弱，供给恢复反而意味着"未来更便宜"（因为进口更通畅）。

6 中国"去库周期"与全球"补库周期"的背离

全球（WTI/Brent定价）中国（SC定价）
库存状态极低（OECD 4-6周缓冲）高位（港口+保税区）
库存方向补库→买近月→近月涨去库→卖近月→近月跌
期限结构Backwardation（合理）Contango（合理）
便利收益正（现货溢价→持有库存赚钱）负（现货折价→持有库存亏钱）

核心洞察：SC的Contango不是"错误定价"，而是中国本地供需的真实反映。便利收益理论告诉我们：当市场持有现货比持有期货更有价值时→Backwardation；当持有现货是负累时→Contango。中国炼厂目前不想持有现货原油（因为加工就亏钱），所以SC近月折价、远月相对稳定。

7 伊朗6.20重新封锁霍尔木兹 → 全球Backwardation加深，SC不受影响

6月16日美伊签署停战备忘录 → 市场预期海峡开放 → Brent一度从$93跌至$79。但6月20日伊朗重新封锁海峡 → WTI/Brent快速反弹至$75/$80。每一次地缘冲击都强化了WTI/Brent的Backwardation（近月受冲击更大，涨得更快），但SC因为：

  1. 中国通过俄罗斯/中亚管道+海运多渠道进口，对霍尔木兹依赖度低于日韩欧
  2. 中国炼厂已经低开工，原油需求本身就弱，运输中断影响被稀释
  3. SC参与者以国内投机+套保为主，对中东地缘敏感度低于CME/ICE

结果就是：每一次海峡危机 → WTI/Brent近月跳涨 → Backwardation加深。SC近月受冲击小 → Contango形态不变甚至加深（因为远月反而因全球紧张而走强）。

三、LU（低硫）& FU（高硫）期限结构深度分析

🔍 品种定义澄清：LU = 低硫燃料油（IMO 2020合规，硫含量≤0.5%） · FU = 高硫燃料油（硫含量3.5%，非脱硫塔船不能使用）
LU偏紧逻辑：航运+IMO合规+柴油挤占调和原料→供给持续偏紧 → 近月升水严重（Backwardation）
FU偏弱逻辑：Jafurah气田替代+供应恢复+IMO限制 → 需求结构性下降 → 近月相对乏力（温和Backwardation）

LU（低硫燃油）：极端Backwardation（仓单=0 + 多维度供给收缩）

LU2608: 3963 → LU2701: 3690，跌幅-273元/吨（-6.9%）

六大驱动（新增科威特断供+LNG分流）：

  1. 核心驱动力 仓单=0吨（极紧！）：INE仓库零仓单，空头无法交割→近月逼仓风险极高。LU2607即将交割，仓单为0意味着所有空头必须平仓→近月价格有硬支撑。
  2. 新增 科威特LSFO发货持续为0万吨（华泰期货6/21）：科威特Al-Zour炼厂（全球最大LSFO出口来源，正常月出口50-80万吨）已连续数月对亚太零发货→LSFO供给出现区域性断供。

FU（高硫燃油）：温和Backwardation + 结构性利空叠加

FU2609: 3101 → FU2706: 2996，跌幅-105元/吨（-3.4%）

  3. IMO合规限制：无脱硫塔船舶只能用LSFO，脱硫塔安装率仅~17%，HSFO需求天花板明确。
  4. 炼厂高开工：全球炼厂为捕捉柴油利润保持高开工→高硫燃料油作为副产品供应增加。

一旦现货回归正常→FU现货价格将大幅下行→期货即便不跟跌也会形成"现货向下回归"的巨大压力。

七、核心结论

  1. SC的"Contango"是真实的但不是深度的：近远月只差9元/桶（1.78%），属于极浅升水，不具备跨期套利条件。七层原因中，中国需求独立坍塌（权重40%）是最根本的驱动力。国泰君安最新数据：地炼开工未来四周继续降至37%→SC Contango有韧性。
  2. 申万宏源关键信号：WTI/Brent期限价差已回到3月初水平→全球Backwardation大幅收窄。SC的极浅Contango和WTI/Brent收敛中的Backwardation正在"对向靠拢"，SC转Backwardation的概率在增大。
  3. FU（高硫燃油）方向确认——结构性看空：
    • 沙特Jafurah气田投产→替代高硫发电 20万吨/月（全球HSFO贸易量2-3%）=结构性永久需求下降
    • 俄罗斯6月发货183万吨+下旬上修→供给压力
    • FU现货基差44-50%严重虚高→期货补涨概率极低，现货向下回归风险大
    • 华泰期货：FU周跌14.44%，走势远弱于LU（-10.04%）
  4. LU（低硫燃油）方向确认——结构性看多：
    • 仓单=0吨（INE仓库零库存）→空头无法交割→近月逼仓风险持续
    • 科威特Al-Zour炼厂LSFO发货持续为0万吨（华泰6/21）→区域性断供
    • 新加坡5月船燃454.8万吨环比+4.5%→需求强劲
    • 柴油高价挤占调和原料→供给间接收缩
  5. 最优策略排序（机构报告印证）：
    1. 多LU+空SC裂解做阔（期限结构双重共振，确定性最高）⭐⭐⭐⭐⭐
    2. 多LU+空FU高低硫做阔（四家机构一致看扩，科威特断供是催化剂）⭐⭐⭐⭐
    3. LU跨期正套（仓单=0+科威特断供逼仓逻辑强化）⭐⭐⭐⭐
    4. 等SC Contango转Backwardation后做正套（申万宏源信号：全球Backwardation在收窄）⭐⭐⭐
    5. FU-SC裂解均值回归（但FU现货虚高44%降低确定性）⭐⭐⭐
  6. 最重要的观察窗口：
    • 中国地炼开工率是SC转折的"先行指标"（当前55-60%→目标65%+）
    • 科威特Al-Zour发货恢复是LU走势的"关键转折"（当前0→恢复日期未知）
    • FU现货基差回归是空FU策略的"利润释放信号"（当前44%→正常5%）
