1. 合成橡胶是什么 — 石化产业链的一环

合成橡胶分类与定位

品种原料中国产量(万t/年)期货标的核心用途
顺丁橡胶(BR)丁二烯（单体聚合）~120BR（上期所）轮胎胎面（70%）+ 鞋底 + 胶管
丁苯橡胶(SBR)丁二烯+苯乙烯~160无期货轮胎胎体（50%）+ 输送带
丁基橡胶(IIR)异丁烯+少量异戊二烯~30无期货内胎/气密层（轮胎）

核心结论：合成橡胶家族中，顺丁橡胶(BR)是唯一有期货的品种（上期所2023年7月上市），也是我们研究的核心标的。丁二烯是所有合成橡胶的共性原料——谁控制了丁二烯，谁就控制了合成橡胶的成本。

🔴 合成橡胶的本质定位：石化产业链的"副产品经济"。丁二烯是乙烯裂解的副产品——它的产量不由自身需求决定，而由乙烯装置的开工率决定。当原油涨价→乙烯利润好→开工率高→丁二烯被动增产→合成橡胶成本下降。这个"副产品"定位决定了合成橡胶的供给极其被动，经常出现"越亏越产"的异常现象。

📐 BR-丁二烯价差：顺丁橡胶加工利润的"晴雨表"

BR期货价格与原料丁二烯的价差，是判断顺丁橡胶行业景气度最直接、最灵敏的指标。它直接反映独立顺丁装置（无配套丁二烯产能）的加工利润状况。

2. 当前价差：1,785元/吨 → 深度亏损，近年最低

BR主力13,185元/吨 丁二烯山东11,400元/吨 BR-丁二烯价差1,785元/吨 vs 2,000亏损线低于215元

三重含义：

  1. 已进入"亏损区间"（<2,000元/吨），且处于近年最低位
  2. 独立顺丁装置（无配套丁二烯）全面亏损——这些装置的丁二烯需要外购，价差就是它们的全部毛利，1,785元/吨的价差完全不足以覆盖辅料、人工、折旧和能耗
  3. 高成本装置已停车或计划停车：威特化工、振华石油、锦州石化等独立装置的开工率已在低位或停产边缘

3. 价差收窄的自我纠正机制

BR-丁二烯价差收窄至亏损区间后，存在自发的供给侧纠正机制：

价差<2,000 → 独立装置亏损 → 高成本装置停车
→ 顺丁橡胶供给减少 → BR价格相对丁二烯走强
→ 价差修复 → 回到2,000-2,500区间

这也是当前BR"跌不动"的核心原因——价格已低于成本，停产是最好的选择。供给端正通过被动收缩来消化过剩，这个过程虽然缓慢但方向确定。

4. 丁苯橡胶(SBR) — 合成橡胶的另一半

丁苯橡胶(SBR)是产量最大的合成橡胶品种（~160万吨/年 vs 顺丁~120万吨），但无期货品种。其原料是丁二烯 + 苯乙烯（比例约75:25），苯乙烯价格跟随纯苯/原油产业链。

与顺丁的对比：

维度顺丁橡胶(BR)丁苯橡胶(SBR)
原料成本构成丁二烯 100%（单体聚合）丁二烯 ~75% + 苯乙烯 ~25%
期货标的✅ BR（上期所）❌ 无
轮胎用途胎面（耐磨、低滚阻）胎体（抓着性、湿滑性能）
替代天胶能力中（耐磨性不如NR）强（与NR物理共混性最好）
当前价格13,620元/吨（全行业亏损）~13,000-13,500元/吨

关键投资含义：BR期货反映的是顺丁橡胶，但轮胎厂配方中同时使用NR+BR+SBR。当研究"天胶vs合成胶替代"时，需要同时看NR/BR/SBR三个品种的价差，而非仅看NR-BR。

关键量化：半钢胎配方中合成胶占比20-30%，当前NR-BR价差3,000-4,000元/吨意味着每吨替换用合成胶可节省3,000-4,000元。对于一个年产500万条半钢胎的工厂（每条用胶3-4kg），将合成胶掺比从25%提升到30%，节省成本约750万-1,000万元/年。

但替代有极限：合成胶不能完全替代天然橡胶。NR的撕裂强度、动态疲劳性能是SR难以替代的——尤其在载重子午胎（全钢胎）中，NR占比不可能低于40%。替代主要发生在半钢胎的胎面胶中（BR含量可适当提高）。

📐 替代边界的完整框架：从价差信号到需求约束

1. 正常掺用比例基线

在轮胎配方中，天然橡胶（NR）和合成橡胶（BR/SBR）处于一个动态平衡状态。正常情况下，乘用车轮胎（半钢胎）和载重轮胎（全钢胎）中的NR与合成橡胶掺用比例如下：

轮胎品类NR正常占比合成胶正常占比可调节弹性关键约束
半钢胎（乘用车）40-55%45-60%±10pp胎面胶可多掺BR，胎体不可
全钢胎（载重）65-80%20-35%±5ppNR撕裂强度不可替代，合成胶最多到35%
工程胎/巨胎90%+<10%极小合成胶几乎不用

2. 替代触发临界点：2,000元/吨价差

当NR价格高于BR/SBR超过2,000元/吨时，轮胎厂的成本核算模型开始触发"配方调整信号"——增加合成胶掺用比例、减少天然橡胶用量。这是行业经验的"替代临界点"。

NR-BR价差区间替代行为对需求的影响
< 1,000元/吨按正常配方生产，无调整动力无影响
1,000-2,000元/吨轮胎厂开始关注，小幅预调NR需求边际-2~3%
2,000-3,000元/吨触发积极调整配方，系统性地增加合成胶掺比NR需求边际-5~8%
> 3,000元/吨最大化合成胶替代，仅在不可替代配方中保留NRNR需求边际-10~15%
但无法再增加

3. 当前价差判断：3,900元/吨 → 远超临界点

RU主力~17,400元/吨 BR主力~13,500元/吨 NR-BR价差~3,900元/吨 vs 2,000临界点超出+1,900元/吨

核心判断：当前价差3,900元/吨已远超2,000元/吨的替代触发临界点。轮胎厂正处于"最大化合成胶替代"区间。这意味着天胶价格在17,500以上的区域，处于"高价抑制需求"的区间——轮胎厂采购意愿明显减弱，每多采购一吨天胶就多承担约3,900元的额外成本。

这也解释了为什么当前胶价（17,400）虽然在厄尔尼诺题材下有一定上行预期，但17,500以上时轮胎厂采购意愿显著下降——不是停产，而是"能不用天胶就不用"的配方策略，边际需求被大量挤出。

4. 替代上限：15-20%的硬约束

合成橡胶不能无限替代天然橡胶。物理性能的鸿沟决定了完全替代的上限不超过15-20%：

性能维度天然橡胶(NR)合成橡胶(BR/SBR)替代后果
撕裂强度极高（结晶自补强）低（无应变结晶）合成胶过多→轮胎易撕裂
动态疲劳性能优良（低生热）差（高滞后→高生热）合成胶过多→高速行驶生热爆胎
抗湿滑性中SBR优秀SBR在胎面可部分替代NR
耐磨性中BR优秀BR在胎面可部分替代NR
加工性好（可塑性强）差（需塑解剂）完全合成胶配方加工困难

NR的撕裂强度和动态疲劳性能源于其应变诱导结晶效应——NR在拉伸时分子链会自发排列成晶体，大幅提高强度。这是所有合成橡胶无法复现的物理特性。因此：

  • 全钢胎中NR占比不可能低于40%——载重行驶条件下生热和撕裂风险太高
  • 半钢胎中NR占比不可能低于30%——胎体帘布层必须用NR保证强度
  • 替代主要发生部位：胎面胶（BR可适当增加）和胎侧胶（NR/BR共混）
  • 总替代上限：NR需求最多被替代15-20%，超出此比例将导致轮胎性能严重劣化
📊 投资含义：NR-BR巨大价差虽然抑制了天胶的边际需求（15-20%的NR消费量被合成胶替代），但80-85%的NR核心需求因物理性能的不可替代性而被"锁定"。这意味着：
  1. 高价会抑制边际需求 → RU在17,500以上缺乏轮胎厂采购支撑，需要有投机资金或天气事件才能突破
  2. 但不会消灭核心需求 → NR在16,000-17,000有极强的需求底部（核心配方必须用NR）
  3. 合成胶替代是"缓冲垫"而非"替代品" → NR和BR最终会通过价差收敛达到均衡，NR不会跌到合成胶的价格

7. 原油传导链 — 从原油到合成橡胶的六步传导

原油（$90.54/桶 WTI） → 石脑油（裂解）
→ 乙烯装置裂解 → 丁二烯（副产品）
→ 单体聚合 → 顺丁橡胶(BR)
→ 轮胎厂配方 → 橡胶制品
传导环节传导效率时滞当前状态
原油→石脑油高度跟随（r≈0.9）0-1周美伊签约→油价暴跌→石脑油跌
石脑油→乙烯→丁二烯中（副产品逻辑）2-4周日韩装置恢复→丁二烯供给增加
丁二烯→顺丁橡胶强（成本占比~75%）1-2周丁二烯跌→BR成本下移
BR→轮胎定价弱（轮胎厂议价能力强）2-4周轮胎厂压价能力增强

关键理解：BR的下跌逻辑不同于天胶。天胶下跌的逻辑是"供应增加+需求减弱"，而BR的下跌逻辑是"原料（丁二烯）跌价→成本下移→估值重塑"。BR价格的锚是丁二烯，丁二烯的锚是原油——因此研究BR，本质上是在研究原油→乙烯→丁二烯的传导。

自4月1日以来累计跌幅24.1%，远超天然橡胶（同期跌幅不足5%）。核心原因：合成橡胶的"先涨后跌"模式——冲突爆发时BR是"最敏感"的橡胶品种（原油直接传导），但冲突缓解时也是"最脆弱"的（丁二烯被动增产+成本崩塌）。

📊 当前估值判断

BR期货基本面静态估值区间为13,200-14,500元/吨（国泰君安）。当前价格13,185元/吨已跌破估值下沿，且低于最低完全成本（13,400元/吨）。

下方支撑逻辑（三层）：

  1. 成本支撑：BR已低于最低成本 → 高成本装置（独山子/振华/威特）停车→供应收紧
  2. NR-BR替代支撑：NR-BR价差3,000-4,000元/吨 → 轮胎厂加大BR采购→需求边际改善
  3. 丁二烯下方空间有限：丁二烯港口库存37,600吨（中性偏低），日韩装置恢复进程是渐进式

上方压力（双层）：

  1. 原油压制：美伊签约→油价中枢系统性下移→丁二烯估值承压
  2. 供应恢复：锦州石化刚开车、产能利用率提升至69.26%→供应边际宽松

结论：BR大概率在13,200-14,500区间震荡筑底。跌破13,000的可能性较低（成本支撑+NR替代支撑），但反弹至14,500以上的空间也有限（原油压制）。

来源：国泰君安期货 2026.06.07（静态估值13,200-14,500）

4. 关键风险点：原油。若中东局势再度恶化→原油暴涨→丁二烯暴涨→BR反弹→策略需要重新评估。但当前美伊签约背景下，原油中枢下移是基准情形。
