一、印度：双压夹击的供给侧断裂

核心命题：全球第二大食糖出口国（占全球贸易量10%）将在至少三个榨季内"几近无糖可出口"——甚至可能首次重启进口（2016-17以来首次）。这不是周期性减产，而是结构性供需断裂。

压力一：厄尔尼诺 — 季风降雨11年最低

指标数据评级
季风降雨11年来最低水平🔴 极负面
6月降水量较均值偏低逾40%🔴 极负面
本榨季产量预期预测3095万 → 下调至2790万吨（-9.9%）🔴 大幅下调
产量 vs 消费2790万 vs 2850万 → 缺口-60万吨🔴 产不足需
农民行为大规模转种大豆/木豆等低耗水作物🔴 远期拖累
主要影响区域马哈拉施特拉邦（最大产糖邦）🔴 核心区冲击
2027-28远期影响种植面积和供应量将受拖累🟡 预期恶化
⚠️ 关键机制 — 农民改种是不可逆的：
马哈拉施特拉邦Sangli地区的农民已经取消甘蔗种植计划，改种大豆。甘蔗是多年生作物（种植一次收获3-4年），一旦农民翻掉宿根蔗改种一年生作物，要恢复到原有甘蔗种植面积至少需要2-3年。这意味着即使2027年季风回归正常，2027-28和2028-29榨季的甘蔗供应仍然受限。

政策/措施具体内容对糖的冲击

乙醇对印度糖市的颠覆性：印度政府已经把乙醇从"辅助性需求"提升为"战略性需求"——灵活燃料汽车的推广意味着乙醇需求不受油价波动影响（不像巴西，巴西乙醇经济性依赖原油价格）。印度乙醇需求从120亿升→300亿升的增长轨迹≈需要额外分走2000-3000万吨甘蔗——这足以抹消印度所有出口潜能。

⚠️ 极端情景 — 印度重启进口：路透社和MEIR Commodities董事总经理Rahil Shaikh指出，如果厄尔尼诺导致种植面积大幅萎缩，印度可能被迫重启食糖进口——将是2016-17和2017-18榨季以来的首次。全球大型贸易商的印度业务团队已开始向总部预警。2009年和2010年印度大规模进口曾推动全球糖价飙升至此前水平的近三倍。

二、全球供给面：印度缺席后的"三缺一"牌局

巴西：史上最矛盾的生产季 — 压榨创纪录，制糖意愿却创新低

指标2026/27（截至5月1日）同比变化

制糖比例38.16%-7.07pp（从45.23%降至38.16%）

核心悖论：巴西2026/27甘蔗预计7.091亿吨（Conab，+5.3%），压榨进度比去年快了74.58%——但糖产量仅增55%，乙醇产量增72%。因为制糖比从45.23%暴跌至38.16%。三分之二的甘蔗被用于乙醇而非制糖。

原因：（1）国际原油价格高企 → 乙醇对糖存在溢价（无水乙醇19.73￠/lb vs ICE糖~14.3￠/lb）→ 糖厂主动多产乙醇；（2）巴西同样推行高乙醇混合政策（E85）→ 国内乙醇需求旺盛。Green Pool：预计巴西中南部全年制糖比仅45%（此前估计48.1%）。

厄尔尼诺风险叠加：USDA指出，强厄尔尼诺预计从8月起给巴西中南部带来超常降雨 → 9-10月加剧 → 可能严重干扰甘蔗采收作业，导致数十家工厂提前结束榨季。S&P Global Energy预测5月上半月产量同比-14%。

⚠️ 为什么四家分歧如此巨大？
分歧的核心在于三个变量的处理方式不同：
（1）印度的出口假设：USDA的480万吨盈余可能仍假设印度会出口一定量糖，而Green Pool/StoneX假设印度=0出口。
（2）巴西的制糖比：Green Pool假设45%，但如果4月实际仅38.16%，全年均值可能低于45%，供给会更紧。
（3）厄尔尼诺量化：USDA对厄尔尼诺影响的量化可能偏保守（印度6月降雨已-40%，远超模型预期）。
→ 市场当前定价的可能是"USDA场景"（盈余480万），但实际在走向"Green Pool场景"（缺口430万）。这个预期差的修正=糖价潜在的上行动力。

七、核心结论

  1. 印度不是减产——是"消失"：厄尔尼诺（甘蔗减产）+ 乙醇（甘蔗分流）+ 农民改种（不可逆的结构性变化）→ 全球可贸易糖供给减少680万吨/年（过去五个榨季年均出口量），相当于全球贸易量的10%。这个量至少三年不会回来。
  2. 巴西"补不上"这个窟窿：虽然甘蔗产量创纪录（7.09亿吨），但62%的甘蔗正在流向乙醇而非制糖。因为原油高企 + FFV推广 + 乙醇溢价。巴西增加的甘蔗绝大部分变成了乙醇，不是糖。
  3. 泰国和欧盟在同步"掉链子"：泰国-15%（1020万吨）、欧盟-12.5%（1530万吨），两家合计减产约405万吨。加上印度缺席的680万吨，三大产区合计从全球市场抹去1085万吨潜在供给。
  4. USDA的480万吨盈余是"最乐观"场景——很可能被下修：USDA的盈余480万吨可能仍暗含印度部分出口的假设。如果印度Q4库存跌破300万吨→USDA会在后续报告中大幅下修→这是糖价的主要"预期差"驱动力。Green Pool已经给出430万吨缺口的判断（与USDA差了910万吨，是近十年来最大的机构分歧）。
  5. 白糖已超越棕榈油成为厄尔尼诺框架下"第一确定性"品种：原因——（A）无长滞后（甘蔗是一年生+当年体现减产）、（B）印度库存告急在眼前（Q3库存仅350万吨→Q4可能跌破300万）、（C）政治敏感性更高（印度政府优先保障国内供应→出口禁令是政治刚需）、（D）巴西的"增产不增糖"矛盾已经确认。
  6. 最重要的观察节点：
    • 2026年7-8月 — 印度季风实际降雨 vs 预测 → 决定甘蔗减产幅度
    • 2026年9月 — 印度禁令到期是否延续 → 大概率延续
    • 2026年10月 — 新榨季开启，印度期初库存实际数字（预测350万吨）
    • 2027年Q1 — 印度库存是否跌破300万 → 进口讨论是否出现
一句话总结：全球糖市正从"过剩叙事"向"赤字叙事"进行结构性切换，而印度是这场切换的"扳机"。680万吨/年的供给消失 + 巴西"增蔗不增糖" + 泰国/欧盟同步减产 = 全球糖价面临至少2-3年的上行周期。当前ICE原糖14¢/SR 5300-5500的价格，尚未充分反映这个结构性转变。
