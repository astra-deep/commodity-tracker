一、豆粕是什么？——从大豆到豆粕的一堂课

🧬 豆粕 = 大豆榨油后的"渣"

大豆被压榨后产生两种产品：豆油(约18-20%) 和 豆粕(约78-80%)。豆粕是提取豆油后的副产品，但它不是"废料"——恰恰相反，豆粕是全球最重要的植物蛋白饲料原料，占全球蛋白粕贸易的70%以上。

豆粕的核心价值 = 蛋白质含量43-48%。作为对比，玉米仅有8-10%。在现代化养殖中，豆粕几乎是无法被完全替代的蛋白来源。

二、当前市场全景：供强需弱，价格承压

⚠️ 核心判断：弱现实主导，强预期待验证

6月豆粕市场呈现典型"供过于求"格局：现货跌破2,800元/吨关口，基差深度走弱至M2609-200元/吨，为近年来少有的深度贴水。上游胀库风险加大，下游采购极为谨慎。

变量③：豆粕库存——胀库风险正在加速

🚨 豆粕库存演变路径（极度危险曲线）

上周库存：约60万吨 → 本周预计：70万吨 → 7月中旬：可能突破100万吨

胀库风险含义：当油厂豆粕库存超过存储能力，会出现被迫停机、降价抛售、甚至"穿库"（以极低价格向饲料厂倾销）。历史上豆粕胀库期通常伴随期货价格暴跌5-10%。

变量④：CBOT美豆——成本锚定

CBOT大豆7月合约跌至1,108.25美分/蒲式耳（约折合人民币2,950元/吨到港成本），创四个月新低。基金多头持续离场。

美豆走弱的三层原因：

  • ① 中西部天气理想：产区降雨和温度适宜，利于大豆生长，天气炒作乏力
  • ② 巴西出口竞争：巴西大豆出口强劲，抢占全球市场份额
  • ③ 中国需求放缓：2026/27年度进口预测从10,330万吨下调至9,550万吨
变量⑤：北美天气——Q3最关键的"扰动项"

市场核心驱动将从"南美供应"切换至"北美天气"。但当前中西部天气对大田作物生长极为有利，这也是CBOT大豆持续走弱的直接原因。

但是：7-8月是大豆关键结荚期，如果出现持续高温干旱（厄尔尼诺背景下概率不低），可能引发快速反弹。这是当前"弱现实"背后唯一的"强预期"锚点。

变量⑥：政策变量——中美贸易与中国采购

中国采购美国大豆的传闻在6月23日获部分证实，为美豆价格提供底部支撑。但采购步伐需要明显加快才能实现全年采购目标。此外，中国降低加拿大菜籽和菜粕关税，将增加替代蛋白供应，间接压制豆粕需求。

四、需求端：三大养殖引擎全面疲弱
引擎①：生猪——最大需求项，但深度亏损中

🐷 生猪产业链对豆粕需求的传导

能繁母猪存栏（决定10个月后生猪出栏量）→ 生猪出栏量 → 饲料消耗量 → 豆粕需求

当前状态：能繁母猪从4,043万头降至3,904万头（2026年5月），仔猪价格17个月来首次下降 → 终端补栏意愿不足 → 6月猪价9.54元/kg（远低于牧原成本11.6元/kg）→ 养殖深度亏损 → 饲料采购极为谨慎

2026年豆粕需求 = 对应2025年4-12月能繁母猪存栏。4-9月存栏在4,000万头左右，10-12月小幅降至3,960万头。全年豆粕需求预计8,500-8,550万吨，同比增幅进一步放缓（2025年增幅约3%，2026年预计不足1%）。

引擎②：家禽——产能高位但效益回落

禽类养殖处于产能高位，但养殖效益明显回落。卓创资讯指出，饲料产量已进入"总量高饱和期"，家禽板块对豆粕的边际增量有限。

引擎③：水产——季节性旺季，但体量有限

二季度水产饲料进入消费旺季，对豆粕需求形成一定边际支撑。但水产饲料中豆粕添加比例（约20-25%）远低于生猪饲料（约20-30%）和禽料（约25-30%），且体量占比仅约10%，季节性利好不足以逆转整体需求疲弱。

五、压榨利润：豆粕-豆油跷跷板与油厂博弈

📐 压榨利润 = 豆粕收入 + 豆油收入 - 大豆成本 - 加工费

油厂买大豆、卖豆粕和豆油。压榨利润是驱动油厂开机率的核心变量。压榨利润好 → 开机率高 → 豆粕供应增加 → 豆粕价格承压。

当前压榨利润处于极度低迷状态。原因：

  • 大豆到港成本：巴西CNF升贴水+CBOT大豆=到港成本约2,950-3,050元/吨
  • 豆粕售价：2,868元/吨（现货）
  • 豆油售价：约7,500-7,700元/吨
  • 加工费：约150元/吨
  • 综合测算：压榨利润约-50至+30元/吨，多数油厂处于盈亏平衡或小幅亏损

压榨利润低迷的深层含义：

  • 短期利多豆粕：油厂可能主动降低开机率，减少豆粕供应
  • 中期利空豆粕：油厂必须维持一定开机率来履行大豆采购合同，被迫产出豆粕
  • 核心矛盾："被动压榨"vs"主动减产"的博弈

⚠️ 被迫压榨的逻辑

大型油厂（如中粮、九三、益海）已经锁定了6-8月的大豆进口合同，大豆到港后如果不压榨，仓储成本更高，且占用资金。因此即便压榨亏损，大厂仍会维持较高开机率。这是豆粕在供过于求阶段仍然持续增库的根本原因。

🔗 2026年6月的特殊状况：豆粕和养猪"双杀"

养猪端：猪价9.54元/kg < 成本11.6元/kg → 深度亏损 → 采购饲料极为克制 → 豆粕需求弱

豆粕端：大豆集中到港+胀库风险 → 供过于求 → 豆粕价格承压

负反馈循环：猪亏 → 减少补栏 → 豆粕需求更弱 → 豆粕价格更低 → 但油厂被迫压榨 → 库存继续升 → 价格继续跌

打破循环的关键：生猪产能真正去化到"缺猪" → 猪价反弹至成本线以上 → 养殖端恢复采购 → 豆粕需求回暖

七、Q3展望："弱现实"与"强预期"的激烈碰撞
路径一：弱现实延续（基准情景，概率50%）

6-8月大豆到港持续高位，胀库压力在7月达到顶峰（库存破100万吨），油厂被迫降价抛售，M2609可能下探2,750-2,800区间。CBOT大豆天气炒作持续乏力（降水理想），美豆跌破1,050美分。需求端猪价继续低迷。

路径二：强预期兑现（上行情景，概率30%）

7月中旬后北美出现高温干旱天气（厄尔尼诺），CBOT大豆快速反弹至1,200+美分，带动国内豆粕反弹。叠加8月后养殖旺季备货启动，M2609回升至3,050-3,200区间。中美贸易消息提供额外支撑。

路径三：极端行情（概率20%）

胀库引发连锁反应：油厂大面积停机 → 豆油供应紧张倒逼压榨 → 豆粕库存继续堆积 → 终端恐慌性观望 → M2609下探2,650-2,700，或天气极端干旱导致CBOT飙升至1,300+美分，M2609冲击3,300+。

八、投资策略与风险提示

✅ 偏多策略（天气/减产逻辑）

中信期货观点：豆粕在2,800-2,850区间布局多单，止损2,750，目标3,050-3,150

核心逻辑：6-8月供过于求已被充分定价，9月后北美天气炒作+国内养殖旺季备货+进口预估下降（2026/27进口9,550万吨）三重利多可能共振。

但需注意：当前天气理想、胀库压力仍在加大，左侧风险极大。

⚠️ 偏空策略（胀库/供需逻辑）

当前2,920附近做空，止损3,020，目标2,800-2,750

核心逻辑：6-8月月均千万吨到港确定，胀库路径清晰（60万吨→70万吨→100万吨+），养殖需求全面疲弱，CBOT美豆七连阴未见止跌信号。

风险：天气突变、中美采购消息、技术性反弹。

📊 稳健策略：跨品种对冲

多豆粕空豆油：如果天气炒作启动（利多粕、利空油），豆粕弹性大于豆油

多M2701空M2609：做多远月（北美天气+去产能后需求恢复）vs 近月（胀库压力），远月升水有望扩大

多菜粕空豆粕（注意风险）：中国降低加拿大菜籽关税，菜粕替代效应增强，但豆菜价差已收窄，空间有限
